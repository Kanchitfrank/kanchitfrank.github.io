Bubble KO (KMUTT Game Jam)

Engine: Unity

Team Size: 3 คน

Roles: 3D Modeling, Split-Screen Developer

Responsibilities:

  • ออกแบบและสร้าง โมเดล 3 มิติ ของฉากและวัตถุในเกม ให้เข้ากับสไตล์เกมแคชชวลสดใส
  • พัฒนา ระบบ Split-Screen ให้ผู้เล่น 2 คนสามารถเล่นพร้อมกันบนจอเดียว

  • ฝึกการสร้าง โมเดล 3D low-poly สำหรับเกมแคชชวลที่ต้องการความลื่นไหล
  • เรียนรู้การพัฒนา ระบบ Split-Screen ทั้งด้านกล้อง การควบคุมผู้เล่น และการแสดงผล UI
  • เข้าใจการแก้ไขปัญหา Collision และ Interaction ระหว่างผู้เล่นในระบบ PvP
  • ได้ทักษะการทำงานเป็นทีมในงาน Game Jam ภายใต้เวลาจำกัด ทั้งด้านการประสานงานและการตัดสินใจร่วมกัน
